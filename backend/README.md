# Backend Scaffold

This folder is reserved for the Node.js + Express backend.

Recommended stack:
- Node.js
- Express.js
- TypeScript
- Sequelize ORM
- JWT + RBAC
- MySQL
- Redis

Suggested structure:
- `src/config/`
- `src/controllers/`
- `src/services/`
- `src/repositories/`
- `src/routes/`
- `src/middlewares/`
- `src/models/`
- `src/validators/`
- `src/helpers/`
- `src/utils/`
