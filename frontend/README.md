# Frontend Scaffold

This folder is reserved for the college website and admin dashboard frontend.

Recommended stack:
- Next.js
- React.js
- TypeScript
- Tailwind CSS
- Redux Toolkit / React Query

Suggested structure:
- `public/`
- `src/api/`
- `src/components/`
- `src/pages/`
- `src/layouts/`
- `src/dashboard/`
- `src/services/`
- `src/store/`
- `src/utils/`
